About

This is a Spotify display featuring three mechanical keyboard keys controlling pause/play, play next/increase volume, play previous/decrease volume, and a master switch(SPDT slide switch) for on/off. This is a Starter Project of [Stasis](https://stasis.hackclub.com/dashboard)
